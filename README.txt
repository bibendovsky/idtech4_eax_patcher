Patcher that enables/disables EAX effects on non-EAX4 sound cards for idTech4 engine based games.

Supported games: Doom 3, Prey, Quake 4.

System requirements:
Latest OpenAL redistributable package.

Usage:
  - Run the program within a game folder.
  - Click "Patch" to enable EAX effects or click "Revert" to disable EAX effects.

Unavailable button means there is no data to patch.

Do not forget to switch on an appropriate EAX (HD) option in the game!
